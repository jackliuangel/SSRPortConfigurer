Port Configurer

Also as a demo of Spring Boot, Spring JPA, Spring Security


Login with JSession ID

Description: H2 DB or no DB in backend supported. Able to set port. (Run with profile JSessionH2 or JSession)



Login with JSON Web Token

Description: with Postgresql DB in backend supported, user audit history supported, expiring in 10 minutes. Able to set port (Run with profile JWT)



Login with GitHub OAuth2

Description: with Postgresql DB and user audit history supported. Only authentication but no authorization. So can not set port (Run with profile OAuth2Github)


CORS Demo

Description: only in profile JSessionH2, CORS is enabled to demo.
